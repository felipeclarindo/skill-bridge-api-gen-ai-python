# Skill Bridge Api Gen AI Python

Web Api Rest to use the model created to make personal resumes.

## Routes

(Generation)

- `GET api/generate` - Use the model.

(Docs)

- `GET api/docs` - The swagger docs

(Description)

- `GET api` - The description of the api

## Steps to run

1. Clone the repository:

```bash
git clone https://github.com/felipeclarindo/skill-bridge-api-gen-ai-python
```

2. Enter directory:

```bash
cd skill-bridge-api-gen-ai-python
```

3. Run the jupyter cells in the file `file_turing_generative_ai.ipynb` to train and generate the model located in `src/notebooks`.

4. Run the api:

```bash
python src/api/api.py
```

5. Api is avaible in:

- <http://localhost:8000/api>
